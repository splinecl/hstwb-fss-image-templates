# HstWB FFS Image Templates

Because I was using AmigaOS 3.2.3 on my Amiga 1200, I wanted to use FFS instead of the PFS3 image templates that comes with HstWB.

Just copy the zip file to the HstWB *Images* folder, and it will show up as an option in the image configuration.
